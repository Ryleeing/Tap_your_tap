# Tap_your_tap（test_record_version)
Built an interactive web application that enables users to compose music in a virtual space and automatically generate synchronized animated visualizations.
Allows users to export high-quality videos for entertainment, music MAD production, and animated opening sequences.
Awarded 2nd Place among Outstanding Projects in the Alibaba Frontend Internship Program.

用于在空间中创作音乐，将音乐转化为网页动画可视化，可下载生成视频，可用于解压游玩、音乐mad、开场音乐动画。
阿里前端练习生优秀项目，结课评比第二名。
原创作品署名：廖苓君，李建庆，何诗茹，陈文岗
